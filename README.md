## Primitives

### Point
* Coordinate (x, y)

### Line
* Point 1: (x1, y1)
* Point 2: (x2, y2)

### Triangle
* Point A: (ax, ay)
* Point B: (bx, by)
* Point C: (cx, cy)

### AABB (Axis Aligned Bounding Box)
* Center Point: (cx, cy)
* Half Width: dx
* Half Height: dy

### OBB (Oriented Bounding Box)
* Center Point: (cx, cy)
* Half Width: dx
* Half Height: dy
* Axis 1 Vector: (u1x, u1y)
* Axis 2 Vector: (u2x, u2y)
